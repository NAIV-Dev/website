# Getting Started (Typescript)

## Prerequisites

Make sure you have installed:

* Node.js v20+
* npm v10+
* npx v10+
* VSCode or VSCodium or Antigravity
* **NAIV Developer Tools** extension

## How to Start

Generate project boilerplate

```bash
$ npx @naiv/ts-api -o project-name
```
